# Groupomania

## BACKEND 

Configure database access in config/config.json
(You need to create a database first)

Run `npm install`
Run `npx sequelize-cli db:migrate`
Run `npx sequelize-cli db:seed:all`

Run `node server` to launch the server


## FRONTEND

Run `npm install`
Run `ng serve`
